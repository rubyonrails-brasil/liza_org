class CalculatorCommand < AppCommand
        
  def self.sum(a, b)= a + b
  def self.sub(a, b)= a - b
  def self.mul(a, b)= a * b
  def self.div(a, b)= a / b

  def call(args)
    log "Called #{self}.#{__method__}
    return help if args.empty?
    
    @args = args
    handle_args
    validate_args
    calculate_result
  rescue StandardError => error
    @error = error
    handle_error
  end

  # ...

end
NAME:

  liza calculator

DESCRIPTION:

  Calculates arithmetic operations.

USAGE:

  liza calculator [a] [op] [b]

  where [a] and [b] are numbers

  where [op] is one of the operators:

        +   addition
        -   subtraction
        *   multiplication
        /   division

EXAMPLES:

  liza calculator 1 + 2
  liza calculator 3 - 4
  liza calculator 5 * 6
  liza calculator 7 / 8
