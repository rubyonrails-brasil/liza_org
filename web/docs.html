Usage

Works without package manager or dependencies ðŸ™‚!

There are 4 ways to get started with pico.css:

Install manually

Download Picoand link /css/pico.min.cssin the < head> of your website. 

                  <
                    link
                    rel=
                    "stylesheet"
                    href=
                    "css/pico.min.css">

Install from CDN

Alternatively, you can use unpkg CDNto link pico.css 

                  <
                    link
                    rel=
                    "stylesheet"
                    href=
                    "https://unpkg.com/@picocss/pico@latest/css/pico.min.css">

Install with NPM

                  
                    npm
                    install
                    @picocss/pico

Install with Composer

                  
                    composer
                    require
                    picocss/pico

Starter HTML template:

<!doctype html>
  <html lang="en">
    <head>
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <link rel="stylesheet" href="css/pico.min.css">
      <title>Hello, world!</title>
    </head>
    <body>
      <main class="container">
        <h1>Hello, world!</h1>
      </main>
    </body>
  </html>
